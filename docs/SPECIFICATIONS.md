# AI職業診断アプリ 機能仕様書

## 1. アプリ概要

### 1.1 コンセプト
MBTI性格診断の形式を採用し、ユーザーの性格タイプに最も適したAI時代の新しい職業を提案するWebアプリケーション。

### 1.2 アプリ名（案）
- AI Career Finder
- AIキャリア診断
- AI職業マッチング

### 1.3 ターゲットユーザー
- キャリアチェンジを検討している社会人
- 就職活動中の学生
- AI時代の働き方に興味がある人
- 自己分析を楽しみたい人

### 1.4 トーン & マナー
- カジュアル・エンタメ寄り
- 親しみやすく明るい雰囲気
- 白背景 × パステルカラー

## 2. 診断システム

### 2.1 診断軸（5軸）
| 軸 | 説明 | 選択肢 |
|----|------|--------|
| E/I | 外向型 / 内向型 | Extraversion / Introversion |
| S/N | 感覚型 / 直観型 | Sensing / iNtuition |
| T/F | 思考型 / 感情型 | Thinking / Feeling |
| J/P | 判断型 / 知覚型 | Judging / Perceiving |
| A/T | 自己主張型 / 慎重型 | Assertive / Turbulent |

### 2.2 質問数
- **合計60問**
- 各軸12問ずつ

### 2.3 診断結果パターン
- 16タイプ（MBTI）× 2（A/T）= **32パターン**
- 例: ENFJ-A, ENFJ-T, INFP-A, INFP-T など

### 2.4 スコアリング方式
各質問で選択した回答に応じてポイントを加算し、各軸の傾向を判定する。

```
例: E/I軸の質問
- 「大勢の人といるとエネルギーが湧く」
  - 強く同意 → E +2
  - 同意 → E +1
  - どちらでもない → 0
  - 反対 → I +1
  - 強く反対 → I +2
```

## 3. 画面構成

### 3.1 画面一覧
| 画面名 | パス | 説明 |
|--------|------|------|
| トップページ | `/` | アプリ紹介・診断開始ボタン |
| 診断ページ | `/quiz` | 60問の質問に回答 |
| 結果ページ | `/result/[type]` | 診断結果とAI職業の表示 |

### 3.2 画面遷移図
```
[トップページ]
    ↓ 「診断を始める」ボタン
[診断ページ]
    ↓ 60問回答完了
[結果ページ]
    ↓ 「Xでシェア」ボタン
[X/Twitter投稿画面]
```

## 4. 各画面の詳細仕様

### 4.1 トップページ（`/`）

#### 表示要素
- アプリロゴ / タイトル
- キャッチコピー（例: 「あなたにぴったりのAI職業を見つけよう」）
- アプリの説明文（所要時間: 約10分、60問）
- 「診断を始める」ボタン（CTA）
- パステルカラーのイラストやアイコン

#### インタラクション
- 「診断を始める」ボタンクリック → `/quiz` へ遷移

---

### 4.2 診断ページ（`/quiz`）

#### 表示要素
- プログレスバー（現在の進捗: X/60）
- 質問番号
- 質問文
- 回答選択肢（5段階）
  - 強く同意
  - 同意
  - どちらでもない
  - 反対
  - 強く反対
- 「戻る」ボタン（前の質問に戻る）
- 「次へ」ボタン（次の質問へ進む）

#### インタラクション
- 回答選択 → 自動的に次の質問へ進む（または「次へ」ボタンで進む）
- 最後の質問回答後 → 結果計算 → `/result/[type]` へ遷移
- プログレスバーはアニメーションで進捗を表示

#### UX考慮事項
- 60問は長いため、10問ごとに励ましメッセージを表示
- 途中離脱防止のため、ブラウザバック時に確認ダイアログを表示

---

### 4.3 結果ページ（`/result/[type]`）

#### 表示要素

##### ヘッダー部
- 診断結果タイプ（例: ENFJ-A）
- タイプの愛称（例: 「主人公」）

##### AI職業セクション
- **AI職業名**（例: AIコミュニティマネージャー）
- **職業の説明**（100-200文字）
- **必要スキル**（3-5項目のリスト）
- **将来性**（高/中/低 + 説明文）

##### 適性セクション
- **あなたの強み**（3-5項目）
  - 例: 管理能力が高い
  - 例: コミュニケーション能力に優れる
  - 例: ビジョンを描くのが得意
- **向いている仕事のスタイル**
  - 例: チームをリードする立場
  - 例: 完璧主義で精密な技術を求められる仕事

##### MBTIタイプ詳細
- タイプの特徴説明
- 各軸のスコア（パーセンテージ表示）
  - 例: E 65% / I 35%

##### シェアセクション
- 「Xでシェアする」ボタン
- 「もう一度診断する」ボタン

#### シェア機能仕様
```
シェアテキスト例:
「AI職業診断の結果、私は【AIコミュニティマネージャー】でした！
タイプ: ENFJ-A（主人公）
#AI職業診断 #MBTI」

シェアURL: https://[domain]/result/ENFJ-A
```

## 1. ENFJ（主人公）

### ENFJ-A: AIコミュニティマネージャー

| 項目 | 内容 |
|------|------|
| **職業名** | AIコミュニティマネージャー |
| **説明** | AIツールを活用してオンラインコミュニティを運営・活性化する職業。メンバーのエンゲージメントを高め、AIを使った効率的なコミュニケーション戦略を立案・実行する。 |
| **必要スキル** | コミュニケーション力、リーダーシップ、AIツール活用、データ分析、コンテンツ企画 |
| **将来性** | 高 - AIコミュニティの需要は急速に拡大中。企業のファンコミュニティ運営需要も増加 |
| **強み** | 人を巻き込む力がある、ビジョンを描ける、チームを鼓舞できる |
| **向いている仕事スタイル** | チームをリードする立場、人と関わる仕事、大きな影響を与えられる環境 |

### ENFJ-T: AI教育コーディネーター

| 項目 | 内容 |
|------|------|
| **職業名** | AI教育コーディネーター |
| **説明** | AI時代に必要なスキルを教える教育プログラムを設計・運営する職業。学習者一人ひとりの成長を丁寧にサポートし、AIツールを活用した効果的な学習体験を提供する。 |
| **必要スキル** | 教育設計、カウンセリング、AIツール活用、カリキュラム開発、共感力 |
| **将来性** | 高 - AI教育の需要は今後も拡大。リスキリング市場の成長が追い風 |
| **強み** | 他者の成長を支援できる、細やかな配慮ができる、信頼関係を築ける |
| **向いている仕事スタイル** | 人の成長に寄り添う仕事、丁寧なサポートが求められる環境 |
